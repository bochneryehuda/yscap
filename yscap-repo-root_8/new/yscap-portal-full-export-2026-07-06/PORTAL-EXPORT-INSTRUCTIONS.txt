YS Capital Portal Export Instructions

Place the CONTENTS of this folder at the root of the portal repository or Render service.

Correct root contents include:
- app
- db
- docs
- src
- web
- package.json
- package-lock.json
- render.yaml

Do not place this folder inside web, web/tools, web/portal, or another nested project folder.

Render deployment:
- render.yaml now runs:
  npm install && npm --prefix app ci && npm --prefix app run build
- That rebuilds the live portal bundle into web/portal during deploy.

Manual deployment:
- From this folder root, run:
  npm install
  npm --prefix app ci
  npm --prefix app run build
- Then upload/deploy the full folder.

Important:
- The marketing/dead term sheet generator remains at web/tools/term-sheet.html.
- The live portal product registration now uses the same frozen Standard/Gold pricing engines and supports staff pricing/admin controls, Excel import/export, term sheets, and pre-approval output.
