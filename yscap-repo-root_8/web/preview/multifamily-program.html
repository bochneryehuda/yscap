FAQ

Questions we hear most

What size properties does this cover?+
This program finances 5+ unit properties — multifamily apartment buildings and mixed-use assets with a multifamily component. Smaller 1–4 unit rentals are handled under our DSCR rental program.
How do you decide bridge, DSCR or term?+
It follows where the property is. Assets being repositioned or added value to fit bridge; stabilized, cash-flowing properties fit DSCR; longer holds fit term. Because the lending is asset-based, leverage varies by experience, asset type and scenario.
